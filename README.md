## 📖 Overview

Bowling Stats Tracker is a **feature-rich** and **user-friendly** app designed to help bowlers keep track of their games, scores, and performance over time. It provides a sleek dark-mode interface with real-time score calculations, league and casual game tracking, and insightful statistics.

## 🎯 Features

✅ **Track Casual & League Games** – Store games in separate categories for better organization.  
✅ **Live Score Calculation** – Automatically updates scores, including strikes, spares, and bonuses.  
✅ **Frame-by-Frame Score Input** – Simple UI for entering and adjusting frame scores.  
✅ **Game Breakdown & Stats** – Displays key performance stats like strikes, spares, and average score.  
✅ **Location & Notes** – Save where you played and any additional details about the game.  
✅ **Dark Mode Support** – Modern UI optimized for dark mode.  
✅ **Save & Review Past Games** – Easily navigate through past games and analyze performance trends.

## 🛠 Tech Stack

- **SwiftUI** – Modern UI framework for smooth animations and adaptive design.  
- **AppStorage (UserDefaults)** – Stores game data persistently on the device.  
- **Xcode 15+** – Built and tested on the latest iOS SDK.
